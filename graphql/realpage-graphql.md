# RealPage GraphQL API

RealPage is a multifamily real estate software platform. The API covers property management, leasing workflows, resident screening, online rent payment, maintenance requests, lease documents, and occupancy analytics.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from RealPage REST API
**Documentation:** https://www.realpage.com
